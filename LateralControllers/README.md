# LateralControllers

This folder contains all assets related to the lateral (steering) comfort controllers. These controllers manage lateral acceleration and jerk to optimize passenger comfort during lane changes, curve following, and other lateral maneuvers.

## Contents

- **Models/** — Simulink models and controller implementations for lateral control.
- **Parameters/** — Tuning parameters, gains, and configuration files for the lateral controllers.
- **Logs/** — Simulation logs and recorded data from lateral controller testing.
- **Others/** — Supplementary files, scripts, and utilities related to lateral control.
