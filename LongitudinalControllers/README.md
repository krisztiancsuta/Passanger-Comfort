# LongitudinalControllers

This folder contains all assets related to the longitudinal (speed/acceleration) comfort controllers. These controllers manage longitudinal acceleration and jerk to optimize passenger comfort during braking, accelerating, and car-following scenarios.

## Contents

- **Models/** — Simulink models and controller implementations for longitudinal control.
- **Parameters/** — Tuning parameters, gains, and configuration files for the longitudinal controllers.
- **Logs/** — Simulation logs and recorded data from longitudinal controller testing.
- **Others/** — Supplementary files, scripts, and utilities related to longitudinal control.
